Cisco organizes the exam into 6 knowledge domains. IP Connectivity (Routing) and Network Fundamentals/Access (Switching) make up the vast majority (**65%**) of your total score.

The CCNA 200-301 uses a score range of 300-1000

A minimum passing score is typically 849

Exam Fee: Around $300 (varies on location)

| Domain | Exam Weight | Key Focus Areas |
| :--- | :---: | :--- |
| **1.0 Network Fundamentals** | **20%** | Routers, Switches, Cabling, IPv4/IPv6 Subnetting, TCP/UDP, Virtualization |
| **2.0 Network Access** | **20%** | VLANs, Trunking, EtherChannel, Spanning Tree (STP/RSTP), Cisco Wireless Architecture |
| **3.0 IP Connectivity** | **25%** | Routing Tables, Forwarding Decisions, Static Routing, OSPFv2 (Single Area) |
| **4.0 IP Services** | **10%** | DHCP, DNS, NAT/PAT, NTP, SNMP, Syslog, Quality of Service (QoS) |
| **5.0 Security Fundamentals** | **15%** | Key Threats, Port Security, DHCP Snooping, Dynamic ARP Inspection (DAI), ACLs, WPA3 |
| **6.0 Automation & Programmability** | **10%** | Controller-Based Networks, REST APIs, Data Formats (JSON), Ansible/Terraform, AI Basics |
